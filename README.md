# truck-detection

ML half of a **WSU Transportation Research Group** feasibility study: detect trucks in single-capture
**PlanetScope SuperDove (~3 m)** satellite imagery over Washington freight corridors, via the
**moving-echo** signal — a colour-separated **blue → red → green** streak a moving vehicle leaves because
the sensor captures its bands a fraction of a second apart. The reference method is a Keypoint R-CNN
(Adamiak et al. 2025).

> **The modeling path is being rebuilt from scratch.** The old training / inference / model code lives in
> [`archive/`](archive/) (recoverable reference). The data pipeline below is active and correct.

## Layout

- **`src/`** — the **data pipeline** (active):
  - `export_coco.py` — labels + GeoTIFFs → COCO keypoints + 64×64 chips
  - `inspect_scene.py` — true-colour scene render · `make_road_mask.py` — OSM road mask (utilities)
- **`backend/`** — Flask API, **data-only** (`/api/dataset`, `/api/scenes`) for the console
- **`frontend/`** — React/Vite **data console** (Dataset / Spec tabs)
- **`archive/`** — retired modeling code (training, inference, model registry + cards, weights, model-capable console) — see [archive/README.md](archive/README.md)
- **`docs/`** — see below
- **`data/active/`** — the current working set: `Annotations-RGB.gpkg` (**339 vehicles / 1017 keypoints**, the only data file tracked in git) + `imagery/` (the 8 labeled scenes) + `coco/` (generated chips)
- **`data/cold/`** — archive (gitignored): unlabeled scenes, old previews, old QGIS project
- **`outputs/`** — derived artifacts (gitignored; regenerated by the scripts)

## Quickstart

```bash
python3 src/export_coco.py    # data/active/ labels + imagery -> data/active/coco/ (64×64 chips)
```

Data console: `python3 backend/server.py` (port 8787) + `cd frontend && npm run dev` (port 5173).

Rebuilding the model? The old pipeline + the reference setup (64×64 chips, leave-one-scene-out, F1 ≈ 0.50 to
beat, anchor ranges) is documented in [archive/README.md](archive/README.md) and [docs/MODELING.md](docs/MODELING.md).
